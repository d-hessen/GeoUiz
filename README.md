About the Project

This project is a lightweight android-based trivia game that lets you test your geography knowledge through different types of questions and activities.
It was originally created as part of my high school project, and anyone can try it!

The game includes:

Multiple-choice geography questions

Flags, capitals and map-based tasks (depending on version)

Clean, simple UI

Easy to expand with new questions
